# AI Agents Guide

Welcome to the Drupal Documentation Upgrade project. As an agent working here, you must follow the standards defined below.

## Core Missions
1. **Conservative Crawling**: Never overwhelm Drupal.org. Use the 10-20 page batch limit with 10s delays and 4h cooldowns.
2. **Quality Documentation**: Transform extracted HTML into high-quality, accessible, and structured Markdown for Drupal 11.
3. **Style Adherence**: Follow the project style guide strictly.
4. **Cost Control**: Use local Ollama by default; use paid cloud models only when explicitly requested or when local fallback policy requires it.

## Standards
- [STYLE.md](./STYLE.md): Detailed documentation standards, tone of voice, and structure.

## Context
- The site is published via GitHub Pages at `https://mgifford.github.io/Drupal-docs/`.
- Local processing is done via Ollama by default.
- Cloud models (Copilot/GitHub Models, optionally Gemini) are fallback-only unless explicitly requested.
